# atividade16
é criada a classe Funcionario, que guarda informações básicas como nome, CPF, salário e data de nascimento. Depois, é criada a classe Gerente, que herda de Funcionario e ganha uma bonificação fixa de R$2000,00 no salário, e a classe Atendente, que também herda de Funcionario e recebe uma comissão extra. Por fim, a classe principal Empresa mostra um menu no qual o usuário pode cadastrar um gerente ou um atendente, listar o funcionário cadastrado e consultar o salário de um funcionário informando o CPF. Cada funcionário é guardado em uma lista, o que permite que o programa trate diferentes tipos de funcionário de forma prática e organizada.
